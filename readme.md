# Preonic Keymap For Programmers That Type In Dvorak

This keymap provides an easier layout for programmers and those who type in Dvorak. Those who use Emacs may find it useful that the 'Alt' and 'Ctrl' keys are very easily accessible and that most default key-chords can be hit without much difficulty. 

This is an unofficial keymap for OLKB's Preonic keyboard. For installation, please refer to QMK Firmware's [documentation](https://docs.qmk.fm)

```
- 'SEnter' refers to a tap for 'Enter' and a long hold for 'Shift'
- The 'Lower' and 'Raise' keys were intended to be hit with the edge of the palms to keep the fingers free for other keys
```
