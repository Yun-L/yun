# Preonic Keymap For Programmers That Type In Dvorak

This keymap provides an easier layout for programmers and those who type in Dvorak.

This is an unofficial keymap for OLKB's Preonic keyboard. For installation, please refer to QMK Firmware's [documentation](https://docs.qmk.fm)
